# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_11

### Phase 1: Editorial Infrastructure (Week 1)
- Set up Next.js 14 project with the "Professional Minimalism" theme.
- Build the "Minimal Nav" and "Editorial Container" shell.
- Implement the "Prestige Logo Strip" global component.
- **Goal:** A high-authority shell that establishes immediate credibility.

### Phase 2: Portfolio Engine & CMS (Week 2)
- Set up Sanity.io for `Article`, `Book`, and `Award` content types.
- Build the dynamic `/portfolio` route with category-based article filtering.
- Implement the "Article List" component with metadata-first hierarchy.
- **Goal:** A working archive of the author's reporting history.

### Phase 3: Book Showcase Hub (Week 3)
- Build the "Book Detail" template with high-impact covers and synopses.
- Implement the regional "Buy Now" link logic (US/Global vs. India).
- Create the "Best of Year" acclaim badge components.
- **Goal:** A prestigious visual centerpiece for the author's books.

### Phase 4: Award Wall & Recognition (Week 4)
- Build the "Awards & Fellowships" page with chronological groupings.
- Implement the "Social Proof" strip for the homepage.
- Build the "Press Mentions" grid for secondary accolades.
- **Goal:** Full establishment of the author's global status.

### Phase 5: About & Professional Inquiry (Week 5)
- Build the "About" page with narrative storytelling and professional portraits.
- Implement the minimalist "Professional Inquiry" form with Supabase storage.
- Final SEO and Page Speed audit for elite client accessibility.
- **Goal:** A complete, world-class journalist and author platform.
