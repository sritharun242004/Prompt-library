# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_15

### Phase 1: Gateway & Shell (Week 1)
- Set up Next.js 14 project with the "Minimalist Publishing" theme.
- Build the "Welcome" landing page with persistent email capture logic.
- Implement the "Sticky Nav" and the "Unified Feed Shell."
- **Goal:** A high-conversion entry point for the publication.

### Phase 2: Content Engine & Multi-mode Feed (Week 2)
- Set up Sanity.io for `Post`, `Publication`, and `Section` content types.
- Build the dynamic homepage with "Magazine" (Grid) and "Newspaper" (List) modes.
- Implement the MDX post renderer with 720px reading container.
- **Goal:** A working archive with flexible layout options.

### Phase 3: Paywall & Tier Logic (Week 3)
- Build the `PaywallCTA` component with the linear-gradient fade overlay.
- Implement the server-side access check for `visibility: paid` posts.
- Set up the "Free Trial" logic and the member signup multi-step flow.
- **Goal:** Locked content is secured and conversion triggers are active.

### Phase 4: Membership & Community (Week 4)
- Integrate Stripe for multi-tier subscription billing and proration.
- Build the "Community Notes" social feed and integrated comment threads.
- Implement the "Member Dashboard" for account and billing management.
- **Goal:** Functional recurring revenue and community engagement.

### Phase 5: Gifting & Final Optimization (Week 5)
- Build the "Gift a Subscription" flow with custom digital cards.
- Integrate Algolia for full-text archive searching.
- Final SEO, meta-data, and image-priority audit for social discovery.
- **Goal:** A production-ready, world-class independent publishing platform.
