# 04 — Build Plan
## Roadmap & Milestones · bw_platform_07

### Phase 1: Global Hub & Routing (Week 1)
- Set up Next.js 14 project with the "Michelin-Grade" theme and Didot font.
- Build the "Global Switcher" split-screen hub with city-routing logic.
- Implement the "Prestige Nav" (Sticky top-bar with city dropdown).
- **Goal:** A prestigious entry portal for the global group.

### Phase 2: Location Sub-sites & CMS (Week 2)
- Set up Sanity.io for `Location`, `Menu`, and `Accolade` structured content.
- Build the dynamic `/(locations)/[city]` template with unique heros.
- Implement the "Accolade Strip" global component.
- **Goal:** Users can portal into specific flagships and see local status.

### Phase 3: Tasting Menu Engine (Week 3)
- Build the visual-first `TastingMenuGrid` with high-end food photography.
- Implement "Chef's Note" tooltips and seasonal badges.
- Create the menu category navigation (Chef's Tasting, Lite, Bar).
- **Goal:** Immersive culinary discovery for each location.

### Phase 4: Policy-Driven Booking (Week 4)
- Build the "Reservation & Policy" interface with strict acceptance logic.
- Integrate OpenTable per-location widget routing.
- Implement the "Private Events" multi-pillar inquiry form.
- **Goal:** Functional, high-standards reservation management.

### Phase 5: Prestige Shop & Polish (Week 5)
- Build the minimalist e-commerce grid for gift vouchers and cookbooks.
- Final luxury animation polish (hard cuts and brass-line expansions).
- Final SEO and 4G optimization for international travelers.
- **Goal:** A complete, world-class restaurant digital platform.
