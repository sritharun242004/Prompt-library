# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_14

### Phase 1: Editorial Infrastructure (Week 1)
- Set up Next.js 14 project with the "Analytical Modernism" theme.
- Build the "Typography Engine" (Global Tiempos/Graphik variables).
- Implement the "Sticky Nav" and "Daily Feed Container."
- **Goal:** A professional news shell with perfect high-contrast legibility.

### Phase 2: Nutgraph & Story Engine (Week 2)
- Build the `Nutgraph` signature summary component.
- Set up the Sanity CMS for `Story` data with bullet-point fields.
- Implement the chronological homepage feed with caricature support.
- **Goal:** Users can read the daily deep-dive and its summary.

### Phase 3: Visual Story Slide Engine (Week 3)
- Build the `SlideDeck` component using Framer Motion (horizontal scroll).
- Implement the progress indicator and mobile-first gesture controls.
- Create the visual-story-specific CMS schema with JSON-based slide arrays.
- **Goal:** A functional, interactive visual storytelling layer.

### Phase 4: Audio Hub & Podcasts (Week 4)
- Build the "Podcast Archive" grid and the sticky audio player bar.
- Integrate Mux or Vimeo for audio hosting and metadata.
- Implement the "Play from Nutgraph" logic (starting audio from the summary).
- **Goal:** Integrated audio experience for the executive user.

### Phase 5: Gifting & Final Optimization (Week 5)
- Build the "Gift a Story" link generation and verification logic.
- Integrate Algolia for hierarchical strategic search.
- Final performance and 4G optimization audit.
- **Goal:** A complete, world-class premium business journalism platform.
