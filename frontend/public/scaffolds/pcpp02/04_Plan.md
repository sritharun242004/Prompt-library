# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_02

### Phase 1: Sidebar & Shell (Week 1)
- Set up Next.js 14 project with the "Clean Utility" theme.
- Build the "Fixed Sidebar" with nested navigation logic (Zustand state).
- Implement the "Homepage Shell" with modular block architecture.
- **Goal:** A highly functional navigation core.

### Phase 2: Technique CMS & Routing (Week 2)
- Set up Contentlayer for MDX-based project and technique data.
- Build the dynamic category routing (`/photography/[technique]`).
- Implement the "Gallery Toggle" (Grid vs. Slideshow) component.
- **Goal:** Users can browse work organized by photographic method.

### Phase 3: Lens Museum Archive (Week 3)
- Build the "Lens Museum" grid and technical detail pages.
- Implement the "Spec Table" component with monospace styling.
- Create the "Sample Gallery" with lightbox integration.
- **Goal:** A professional-grade technical gear library.

### Phase 4: Digital Commerce Layer (Week 4)
- Build the "Shop" grid for LUTs, Presets, and Ebooks.
- Integrate Lemonsqueezy or Stripe for instant digital delivery.
- Implement the "Related Tools" logic (linking techniques to presets).
- **Goal:** Automated monetization of creator assets.

### Phase 5: Educational Blog & Polish (Week 5)
- Build the "Tutorial Blog" with high-readability typography.
- Implement "Affiliate Gear" blocks within blog posts.
- Final performance audit and image optimization for saturated content.
- **Goal:** A complete, high-utility hub for experimental photography.
