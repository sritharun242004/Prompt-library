# 05 — Epics & Stories
## Work Breakdown · pcpp_platform_02

### Epic 1: Navigational Structure
*As a visitor, I want a persistent menu so I can always find my way between art and education.*
- **Story 1.1:** Navigate the site via a fixed vertical sidebar.
- **Story 1.2:** Access nested "Technique" sub-menus without leaving the current page.
- **Story 1.3:** See a live cart count and total in the sidebar/footer.

### Epic 2: Technical Discovery
*As an enthusiast, I want to see exactly how an image was made.*
- **Story 2.1:** Filter the portfolio by specific techniques (e.g., Weird Lenses).
- **Story 2.2:** Read 1-sentence técnica summaries on each category page.
- **Story 2.3:** Switch between a thumbnail grid and large-scale slideshow view.

### Epic 3: The Lens Museum
*As a gear collector, I want a structured library of vintage lens samples.*
- **Story 3.1:** Browse a list of lenses categorized by mount or age.
- **Story 3.2:** View a technical spec table for each lens (Aperture, Focal length, etc.).
- **Story 3.3:** See high-resolution sample photos taken with specific lenses.

### Epic 4: Knowledge Commerce
*As a student, I want to purchase the tools the artist uses instantly.*
- **Story 4.1:** Browse a shop of digital assets (LUTs, Presets, Ebooks).
- **Story 4.2:** Complete a checkout in under 3 steps.
- **Story 4.3:** Receive a secure download link via email immediately after payment.
