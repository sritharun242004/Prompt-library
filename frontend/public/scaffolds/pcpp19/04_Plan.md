# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_19

### Phase 1: Discovery Infrastructure (Week 1)
- Set up Next.js 14 project with the "Trust & Success" theme and Inter font.
- Build the "Goal-Selection Search" and the centralized Hub Homepage.
- Implement the "Global Goal" state management using Zustand.
- **Goal:** A scannable portal for 200+ exam goals.

### Phase 2: Exam Sub-sites & Educator CMS (Week 2)
- Set up Sanity.io for `Exam`, `Educator`, and `Batch` content types.
- Build the dynamic `/(exams)/[goal]` template with success-stat strips.
- Implement the "Top Educator" pedigree grid and individual profile views.
- **Goal:** Users can find their specific exam and its best teachers.

### Phase 3: Subscription comparison & Hub (Week 3)
- Build the "Plus vs. Iconic" comparison module with side-by-side feature lists.
- Implement the pricing-tier toggle (Monthly vs Annual) with savings math.
- Integrate Razorpay for initial subscription purchase and mandates.
- **Goal:** Functional high-AOV conversion engine for premium tiers.

### Phase 4: Live Classroom & Interaction (Week 4)
- Build the `ClassroomUI` featuring video player and real-time chat (Supabase).
- Implement the "Raise a Hand" and "Live Poll" pedagogical components.
- Integrate Mux Player for HLS live stream delivery.
- **Goal:** End-to-end live learning experience for subscribers.

### Phase 5: Success Dashboard & Polish (Week 5)
- Build the personal "Study Planner" and "Test Series" analysis dashboard.
- Final performance audit for large-scale grid and live-stream LCP.
- Accessibility audit for high-contrast typography and goal-search clarity.
- **Goal:** A complete, world-class competitive exam preparation platform.
