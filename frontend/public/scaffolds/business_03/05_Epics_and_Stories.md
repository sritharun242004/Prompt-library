# 05 — Epics and Stories
## K-12 and Competitive Learning Platform · business_platform_03

---

### Epic 1 — Proof and Credibility

Story: parents and students can verify platform outcomes before engaging with forms.

Acceptance Criteria:
- proof metrics display year and program context labels
- teacher profiles show subject, experience, and credentials
- results page filterable by exam type, year, and program
- methodology note explains what each metric means

---

### Epic 2 — Program and Pathway Clarity

Story: visitors can identify the right program for their class level and exam goal.

Acceptance Criteria:
- program matrix complete for all class levels (4–12) and exam types
- mode comparison (online / offline / hybrid) visible per program
- duration, schedule, and pedagogy summary present on each track page
- program grid navigable on mobile with no horizontal overflow

---

### Epic 3 — App Experience

Story: parents and students understand the app's learning value before installing.

Acceptance Criteria:
- app feature list explains personalization, live doubt resolution, and adaptive practice
- screen previews captioned and accessible
- app store CTA buttons include accessible label text
- install event tracked with source page context

---

### Epic 4 — Center Discovery

Story: visitors can find their nearest center, view batches, and submit a local inquiry.

Acceptance Criteria:
- city index page lists all available cities
- center page shows contact, address, map, batch schedule, and fee range
- local inquiry form captures city, class, and contact
- center page inquiry tracked as distinct event from homepage lead

---

### Epic 5 — Admissions Conversion

Story: visitors can complete a free class booking or scholarship registration with minimal friction.

Acceptance Criteria:
- free class form collects student name, class level, city, parent contact
- OTP verification completes server-side and confirms before lead submit
- scholarship flow: eligibility → slot selection → center preference → confirmation
- both forms show policy/refund notice before final submit
- lead payload includes source, page context, class level, and exam intent

---

### Epic 6 — Support and Policy Clarity

Story: parents can access support and policy information before submitting a form.

Acceptance Criteria:
- fee policy and refund/cancellation terms accessible within 2 clicks from any form
- FAQ addresses top 10 parent and student questions
- contact path visible with phone/WhatsApp/email options
- escalation path available for post-enrollment queries
