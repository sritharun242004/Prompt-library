# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_01

### Phase 1: Gallery Infrastructure (Week 1)
- Set up Next.js 14 project with a Pure Black theme.
- Build the "Minimal Nav" and "Immersive Layout" shell.
- Implement the "High-Density Grid" for the homepage.
- **Goal:** A scannable project overview that feels premium and dark.

### Phase 2: Project Engine & Narrative (Week 2)
- Set up Supabase with the `projects` and `media` schema.
- Build the "Full-Bleed" image components.
- Implement the "Narrative Scroll" template (Mix of text and media).
- **Goal:** Users can navigate from the grid into a deep project story.

### Phase 3: Director's Theater (Week 3)
- Integrate Mux or Vimeo for high-quality video playback.
- Build the "VideoReel" component with auto-play and full-screen theater mode.
- Implement the "Director" landing page with looping reels.
- **Goal:** A professional motion showcase.

### Phase 4: Fine-Art Shop Integration (Week 4)
- Build the minimalist "Shop" grid and product detail pages.
- Integrate Stripe for a seamless checkout overlay.
- Implement the "Print Edition" tracker (Supabase inventory).
- **Goal:** Users can purchase prints directly from the portfolio.

### Phase 5: Storytelling & Learn (Week 5)
- Build the "Dudes with Cameras" blog and BTS section.
- Implement the "Gear" list with technical monospace styling.
- Final SEO and Performance audit for 4K display support.
- **Goal:** A complete, world-class photography and motion platform.
