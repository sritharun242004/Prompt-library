# 04 — Build Plan

- Set up Next.js 14 project with the "Friendly Tech" theme and Circular/Inter fonts.

- Implement the "Hey Friend" personal brand hero with social-proof strips.

- **Goal:** A professional-grade intellectual headquarters for long-form content.

### Phase 4: Acquisition & Lead Magnets (Week 4)
- Build the high-conversion "LifeNotes" newsletter blocks with integrated proof.
- Implement the "Bonus Unlock" order-verification logic and member portal.
- Create the "Welcome Bonus" automated delivery flows based on user intent.
- **Goal:** Functional high-volume growth engine is live.

### Phase 5: Polish & "Hand-drawn" Annotations (Week 5)
- Implement the custom SVG "Annotation" engine for hand-drawn highlights.
- Final performance audit for large-scale grid and video LCP.
- Accessibility audit for high-contrast colors and "Hey Friend" screen-reader flows.
- **Goal:** A complete, world-class multi-product creator flagship.
