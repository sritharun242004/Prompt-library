# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_10

### Phase 1: Editorial Shell & Typography (Week 1)
- Set up Next.js 14 project with the "Tech Editorial" theme.
- Build the "Headline" and "Body" typography engines using Inter font.
- Implement the "Minimal Sticky Nav" and "Unified Feed Container."
- **Goal:** A professional, fast-loading news-shell.

### Phase 2: Feed Engine & Ghost API (Week 2)
- Set up the Ghost Content API and define the post fetching logic.
- Build the chronological homepage feed with "Headline > Deck" cards.
- Implement "Category Tags" (e.g., Platforms, Policy, AI) filtering.
- **Goal:** A working news-stream from the CMS.

### Phase 3: Paywall & Member Logic (Week 3)
- Implement the "Server-side Access Check" for paid posts.
- Build the "Fade-to-CTA" paywall component with gradient overlays.
- Integrate NextAuth.js or Clerk for "Magic Link" authentication.
- **Goal:** Locked content is protected and teasers are visible.

### Phase 4: Subscription Portal & Stripe (Week 4)
- Build the "Member Portal" modal with plan selection toggles.
- Integrate Stripe Checkout and Customer Portal for billing.
- Implement "Member Status" synchronization between Stripe and Ghost.
- **Goal:** End-to-end subscription conversion is functional.

### Phase 5: Search, Podcast & Polish (Week 5)
- Integrate Algolia for full-text archive searching.
- Build the "Podcast Player" minimal embed blocks.
- Final SEO and Page Speed audit (Focus on text-rendering performance).
- **Goal:** A complete, world-class journalist publication and membership hub.
