# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_09

### Phase 1: Typography & Paper Core (Week 1)
- Set up Next.js 14 project with the "Warm Paper" theme.
- Build the "Typography Engine" (Global font-variables and leading/kerning resets).
- Implement the "Reading Shell" with the 720px centered container.
- **Goal:** A perfect, distraction-free environment for raw text.

### Phase 2: Narrative CMS & Feed (Week 2)
- Set up Sanity.io for `Essay` and `Subject` content types.
- Build the chronological homepage stream with long-form excerpts.
- Implement "Vintage Art" lazy-loading for hero illustrations.
- **Goal:** Users can browse the publication history smoothly.

### Phase 3: Subject Index & Deep Search (Week 3)
- Build the "Subject Index" list and dynamic topic routing.
- Integrate Algolia or Fuse.js for real-time archive searching.
- Create the "Surprise Me" randomizer logic using Sanity GROQ.
- **Goal:** Ease of discovery across 20+ years of intellectual archives.

### Phase 4: Reader-Supported Donation Layer (Week 4)
- Build the "Donating = Loving" module with emotive writer-copy.
- Integrate Stripe for one-time and recurring contributions.
- Implement the "Supporter Status" logic (session-based).
- **Goal:** A functional, ad-free revenue engine.

### Phase 5: Threading & Synthesis (Week 5)
- Build the "Complementary Thread" engine for manual post linking.
- Implement the "Marginalia" sided-notes and parenthetical UI.
- Final SEO and accessibility audit (Focus on screen readers for long-form).
- **Goal:** A complete, world-class literary publication and synthesis hub.
