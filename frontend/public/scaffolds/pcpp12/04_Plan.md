# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_12

### Phase 1: Editorial Shell & Timeline (Week 1)
- Set up Next.js 14 project with the "Modular Editorial" theme.
- Build the "Fixed Profile Header" and the "Work Timeline" vertical engine.
- Implement the "Paper" background theme and Inter font hierarchy.
- **Goal:** A scannable professional identity shell.

### Phase 2: Expandable Layers & CMS (Week 2)
- Set up Sanity.io for `Profile`, `WorkHistory`, and `Project` schemas.
- Build the `ProjectLayer` component with Framer Motion expansion logic.
- Implement the MDX narrative renderer for deep-dive case studies.
- **Goal:** Users can dive into specific work history details.

### Phase 3: Writing Hub & Typography (Week 3)
- Build the "Writing" tab and article list component.
- Fine-tune the "Editorial Serif" typography for long-form essays.
- Implement the `ComplementaryPost` logic (Linking essays to work projects).
- **Goal:** A professional space for thought leadership and case studies.

### Phase 4: Side Projects & Explore Feed (Week 4)
- Build the "Side Project" grid with high-impact image cards.
- Create the "Status Scroller" for real-time creative updates.
- Implement the "Explore" discovery feed with role-based filtering.
- **Goal:** A complete, multi-faceted professional network.

### Phase 5: Team Badges & Final Polish (Week 5)
- Build the "Team Page" template and collaboration badges.
- Implement the minimalist "Account Settings" and "Magic Link" auth.
- Final performance and accessibility audit for high-signal clarity.
- **Goal:** A world-class, living resume and professional platform.
