# 04 — Build Plan
## Roadmap & Milestones · bw_platform_04

### Phase 1: Airy Shell & Nav (Week 1)
- Set up Next.js 14 project with the Coconut/Citrus theme and Montserrat font.
- Build the "Atelier Nav" (Sticky with centered logo) and full-bleed Hero Engine.
- Implement the "Homepage Grid" system with expansive whitespace.
- **Goal:** A prestigious, breezy homepage presence.

### Phase 2: Journal CMS & Content (Week 2)
- Set up Sanity.io for `JournalEntry` and `Collection` structured content.
- Build the dynamic `/journal` index and single-story templates.
- Implement the Spotify/Soundcloud audio embed logic for "Nico Radio."
- **Goal:** Users can consume lifestyle content and listen to playlists.

### Phase 3: Discovery Engine & Pairing (Week 3)
- Build the "Pair it with" and "Shop the Set" cross-category carousels.
- Implement the collection-tagging logic that bridges Fashion and Home items.
- Create the "Shop the Look" interactive points within journal articles.
- **Goal:** Driving multi-faceted discovery and higher AOV.

### Phase 4: Gifting Concierge Hub (Week 4)
- Build "The Gift Shop" landing page with price-point and occasion filters.
- Implement the curated gift advice UI and corporate inquiry form.
- Integrate Algolia for high-speed gift-guide search results.
- **Goal:** A high-conversion gifting destination for premium users.

### Phase 5: Mobile Polish & Performance (Week 5)
- Optimize high-res lifestyle imagery for mobile LCP.
- Implement horizontal "Look" scrollers for small screens.
- Final SEO and Page Speed audit for global-standard performance.
- **Goal:** A complete, world-class lifestyle retail and editorial platform.
