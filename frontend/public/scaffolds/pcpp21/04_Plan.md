# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_21

### Phase 1: Event Shell & Motion (Week 1)
- Set up Next.js 14 project with the "Design-Forward" theme and Inter font.
- Build the "Video Hero" engine and the "Sticky Urgency Ticker" marquee.
- Implement the "Minimalist Hub" layout with dark-mode defaults.
- **Goal:** A high-energy, immersive opening presence.

### Phase 2: Speaker Engine & CMS (Week 2)
- Set up Sanity.io for `Speaker` and `Track` structured content.
- Build the "B&W-to-Color" hover grid and the speaker directory.
- Implement the "Bio Overlay" component for deep-dive speaker info.
- **Goal:** Users can browse the world-class talent lineup.

### Phase 3: Schedule Hub & Timeline (Week 3)
- Build the interactive `ScheduleTimeline` with track-filtering pills.
- Implement the real-time time-zone formatting and date-tabbing logic.
- Create the "My Schedule" Zustand store with local persistence.
- **Goal:** A functional, personalized 3-day conference schedule.

### Phase 4: Logistics & "Commons" (Week 4)
- Build the "Logistics Hub" with high-res venue maps and travel info.
- Implement the "Watch Party" map integration for virtual attendees.
- Create the "FAQ & Help" scannable sections.
- **Goal:** A complete source of truth for event logistics.

### Phase 5: Registration & Final Polish (Week 5)
- Build the multi-tier registration portal and Stripe checkout integration.
- Final performance audit for 4K video heros and heavy image carousels.
- Accessibility audit for high-contrast colors and monospaced text.
- **Goal:** A complete, world-class design event digital platform.
