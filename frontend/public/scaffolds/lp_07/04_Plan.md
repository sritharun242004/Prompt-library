# 04 — Build Plan
## Roadmap & Milestones · lp_platform_07

### Phase 1: Authority Shell & Hero (Week 1)
- Set up Next.js 14 project with the "High-Contrast" theme and Montserrat font.
- Build the "Minimal Nav" (Courses, Books, Newsletter) and the Hero section.
- Implement the "Creator Portrait" layout and massive follower stats strip.
- **Goal:** A bold, prestigious entry point for the brand.

### Phase 2: Failure Resume Engine (Week 2)
- Set up Sanity.io for `FailureLog` structured narrative content.
- Build the vertical "Failure Timeline" component with lesson-learned tooltips.
- Implement the "Emotional Discovery" section below the hero.
- **Goal:** established deep user trust through radical transparency.

### Phase 3: Course Marketplace & Labels (Week 3)
- Set up Sanity.io for `Course` and `WebVeda` content types.
- Build the transparently priced course grid with outcome-led chips.
- Implement the category filtering (Money, Career, Mindset).
- **Goal:** A high-conversion digital marketplace for educational products.

### Phase 4: Book Showcase & Retailer Hub (Week 4)
- Build the 3D book render component and the multi-retailer affiliate hub.
- Implement the "Bonus Unlock" lead-capture modal for book buyers.
- Create the "As Seen On" logo cloud with grayscale-to-color hovers.
- **Goal:** Established global authority and book-sales functionality.

### Phase 5: Acquisition & Search Archive (Week 5)
- Build the "Wanderings" newsletter blocks and integrated archive search.
- Final performance audit for Indian 4G/5G mobile benchmarks.
- Accessibility audit for high-contrast neon colors and timeline structures.
- **Goal:** A complete, world-class Indian creator flagship platform.
