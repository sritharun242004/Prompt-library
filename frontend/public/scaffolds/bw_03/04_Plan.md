# 04 — Build Plan
## Roadmap & Milestones · bw_platform_03

### Phase 1: Homely Shell & Reels (Week 1)
- Set up Next.js 14 project with the Suta Red/Teal theme.
- Build the "Sticky Community Nav" and the horizontal "Reel Engine."
- Implement the "High-Vibrancy" lifestyle hero carousel.
- **Goal:** A high-energy, mobile-optimized landing page shell.

### Phase 2: Story CMS & Suta Book (Week 2)
- Set up Sanity.io for `SutaStory` and `Product` structured content.
- Build the dynamic `/suta-book/[slug]` template with interspersed buy-points.
- Implement the "Evocative Naming" metadata logic.
- **Goal:** Users can read collection backstories and buy within the narrative.

### Phase 3: Offer Engine & Intelligent Cart (Week 3)
- Build the Zustand-powered cart with complex offer logic (Buy 3 Get 1).
- Implement the "Quick-Option" mini-modal for grid-level size selection.
- Create the "Offer Progress Bar" component for the cart drawer.
- **Goal:** A high-conversion cart experience with real-time incentive tracking.

### Phase 4: Community Grid & Social Proof (Week 4)
- Build the "Suta Community" masonry grid for customer photos.
- Set up Supabase for testimonial storage and "Shop the Look" mapping.
- Implement the WhatsApp concierge floating button.
- **Goal:** Deep integration of social proof and personalized help.

### Phase 5: Inclusion & Final Speed (Week 5)
- Finalize the inclusive size range UI ( hidding nothing, disabling OOS).
- Final SEO and Page Speed audit (Focus on Reel LCP and high-res fabric detail).
- Final luxury animation polish (smooth slide-ups and vibrant fades).
- **Goal:** A complete, world-class D2C boutique and community platform.
