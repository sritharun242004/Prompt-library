# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_06

### Phase 1: Luxury Shell & Grid (Week 1)
- Set up Next.js 14 with the "Sophisticated Editorial" theme.
- Build the "Fixed Top-bar" and wide-tracked navigation system.
- Implement the "Masonry Engine" for mixed-media portfolios.
- **Goal:** A prestigious homepage shell with high-end grid performance.

### Phase 2: Journal & Storytelling (Week 2)
- Set up Sanity CMS for `Work` and `Journal` content types.
- Build the "Vertical Story" template with centered editorial captions.
- Implement "Lazy-priority" image loading for long vertical scrolls.
- **Goal:** Users can experience a complete, immersive wedding story.

### Phase 3: Education Hub (Week 3)
- Build the "Education" landing page and event filtering UI.
- Set up Supabase for `events` and `registrations` storage.
- Implement the "Mentorship Application" form with custom validation.
- **Goal:** A functional hub for workshops and global conferences.

### Phase 4: Prestige & Press Polish (Week 4)
- Build the "Prestige Strip" for magazine logos and ambassador status.
- Implement parallax hero effects and "Springy" hover transitions.
- Build the unified "Work vs Team" portal routing.
- **Goal:** A world-class authority signal for the brand.

### Phase 5: Commerce & Final Speed (Week 5)
- Integrate Stripe for masterclass deposit payments.
- Final SEO and Performance audit (Focus on image compression for Retina displays).
- Accessibility audit for high-contrast typography and media labels.
- **Goal:** A production-ready, global luxury photography platform.
