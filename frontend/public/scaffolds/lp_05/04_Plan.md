# 04 — Build Plan
## Roadmap & Milestones · lp_platform_05

### Phase 1: Authority Shell & Nav (Week 1)
- Set up Next.js 14 project with the "Authority Minimalism" theme and Inter font.
- Build the "Atelier Nav" (Sticky with deep category dropdowns) and the Content Hub shell.
- Implement the "White Paper" background theme and 1.7 leading resets.
- **Goal:** A prestigious, high-legibility repository shell.

### Phase 2: Chronological Engine & CMS (Week 2)
- Set up Sanity.io for `Episode`, `Article`, `Guest`, and `Topic` schemas.
- Build the reverse-chronological homepage feed with type-based icons.
- Implement the "Sidebar Hub" with topic clouds and search bars.
- **Goal:** Users can browse 15+ years of content history smoothly.

### Phase 3: Show Notes Builder (Week 3)
- Build the dynamic `/podcast/[slug]` template with integrated audio players.
- Implement the "Meticulous List" component for People/Books/Links.
- Create the "Affiliate Middleware" logic for automated link tagging.
- **Goal:** Resource-dense pages that drive monetization and user trust.

### Phase 4: Curation Hub & "Start Here" (Week 4)
- Build the "Start Here" landing page with logic-based greatest-hits grouping.
- Implement the Algolia search engine for high-speed archive retrieval.
- Create the "Best Of" category carousels for the homepage.
- **Goal:** Low-friction onboarding for new audience members.

### Phase 5: Acquisition & Analytics (Week 5)
- Build the high-conversion newsletter heros and sticky sidebar callouts.
- Implement the "Affiliate Impact" dashboard for internal tracking.
- Final SEO and Page Speed audit (Targeting 100/100 Accessibility).
- **Goal:** A complete, world-class creator brand hub and content ecosystem.
