# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_08

### Phase 1: Studio Shell & Mosaic (Week 1)
- Set up Next.js 14 project with the "Production Modern" theme.
- Build the "Fixed Sticky Nav" and the "Service Mosaic" layout engine.
- Implement the "Video Hero" component with poster-fallback logic.
- **Goal:** A high-energy homepage shell with service pillar access.

### Phase 2: Category & Project Engine (Week 2)
- Set up Sanity.io for `Service` and `Project` structured content.
- Build the dynamic category routing (`/services/[type]`) with statistical headers.
- Implement the "Series Page" that groups 4K trailers and photo galleries.
- **Goal:** Users can browse work organized by production domain.

### Phase 3: Team & Trust Hub (Week 3)
- Build the "Meet the Crew" grid and individual team cards.
- Implement the "Client Testimonial Slider" with verified rating badges.
- Create the "Press Strip" with high-contrast monochrome logos.
- **Goal:** Immediate establishment of studio-scale trust and authority.

### Phase 4: Video Preview Pipeline (Week 4)
- Integrate Mux Player and thumbnail API for high-performance previews.
- Build the "Theater Mode" modal for 4K trailer viewing.
- Implement hover-to-play logic for all project cards.
- **Goal:** A high-utility, motion-first browsing experience.

### Phase 5: Inquiry & Optimization (Week 5)
- Build the multi-pillar inquiry form with event-tracking logic.
- Final SEO and Page Speed audit (Focus on heavy grid LCP optimization).
- Performance audit for international 4G/5G destination browsing.
- **Goal:** A production-ready, high-conversion studio powerhouse.
