# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_05

### Phase 1: Royal Infrastructure (Week 1)
- Set up Next.js 14 project with Gold/Maroon theme and Serif/Sans fonts.
- Build the "Cinema Hero" and "Minimalist Sticky Nav."
- Implement the "Celebrity Strip" and "Universe Overview" sections.
- **Goal:** A grand, high-impact presence for the main landing page.

### Phase 2: Chapter Engine & CMS (Week 2)
- Set up Sanity.io for `Chapter` and `Snippet` structured content.
- Build the dynamic `/stories/[slug]` template with interspersed media blocks.
- Implement the "PhotoGrid" engine with support for diptych and full-bleed layouts.
- **Goal:** Users can experience a complete "Wedding Story" narrative.

### Phase 3: 4K Cinematography Hub (Week 3)
- Integrate Mux for 4K video delivery and player management.
- Build the "Films" landing page with categorized trailers.
- Create the custom "Cinema Overlay" player with "Director's Notes."
- **Goal:** A dedicated, high-performance video showcase.

### Phase 4: Inquiry Portal & Lead-Gen (Week 4)
- Build the "Tell Your Story" multi-step inquiry form.
- Integrate Supabase for lead storage and Resend for auto-responders.
- Implement the "Heirloom Photobook" teaser section.
- **Goal:** A high-conversion lead generation engine.

### Phase 5: Brand Polish & Romantic Motion (Week 5)
- Build romantic transitions (Slow fades and gold-line expansions).
- Final image optimization for high-vibrancy RAW-processed shots.
- Performance and Accessibility audit for high-net-worth mobile users.
- **Goal:** A production-ready, celebrity-grade wedding portfolio.
