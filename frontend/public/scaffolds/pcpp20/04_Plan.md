# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_20

### Phase 1: Corporate Shell & Trust (Week 1)
- Set up Next.js 14 project with the "Professional Growth" theme and Montserrat font.
- Build the "Program Finder" nav and the centralized Marketplace Grid.
- Implement the "University Trust Wall" global logo strip.
- **Goal:** A prestigious, institutional entry point for the platform.

### Phase 2: ROI Visualization & Outcome CMS (Week 2)
- Set up Sanity.io for `University`, `Program`, and `HiringPartner` schemas.
- Build the `OutcomeViz` components with Recharts and count-up animations.
- Implement the "Career Transition" video feed with before/after data points.
- **Goal:** Users can see evidence of career impact immediately.

### Phase 3: Syllabus Lead-Gen Hub (Week 3)
- Build the `ProgramCard` with integrated "Download PDF" lead-capture modal.
- Implement the server-side PDF delivery worker post-lead creation.
- Create the "Domain-based" sub-sites (e.g., `/courses/ai`, `/courses/mba`).
- **Goal:** High-conversion curriculum discovery and lead acquisition.

### Phase 4: Expert Inquiry & Routing (Week 4)
- Build the "Talk to Expert" multi-step form with regional validation.
- Set up the Supabase logic for routing leads to admission experts.
- Implement the "Mentor Spotlight" grid on program landing pages.
- **Goal:** Functional high-touch sales and inquiry management.

### Phase 5: Final Polish & AOV Optimization (Week 5)
- Build the "Enterprise / For Teams" B2B landing page.
- Final performance audit for large-scale grid LCP (Target <1s).
- Accessibility audit for high-contrast data visualization and form labels.
- **Goal:** A complete, world-class professional edtech platform.
