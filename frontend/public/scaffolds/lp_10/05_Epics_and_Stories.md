# 05 — Epics & Stories
## Work Breakdown · lp_platform_10

### Epic 1: Immediate Creation
*As a visitor, I want to start building my form instantly without creating an account.*
- **Story 1.1:** Experience an "Action-First" hero that drops me directly into a blank document.
- **Story 1.2:** Use slash-commands (/) to insert form fields and text elements in under 2 seconds.
- **Story 1.3:** See my form auto-save locally so that I can return to it later without an account.

### Epic 2: Productive Discovery (Templates)
*As a busy builder, I want to start with a proven template for my specific use-case.*
- **Story 2.1:** Browse a categorized library of templates (Marketing, Feedback, HR).
- **Story 2.2:** Preview a functional form and click "Use this Template" to start editing instantly.
- **Story 2.3:** Experience high-contrast "Unlimited" badges that reassure me advanced features are free.

### Epic 3: Transparent Progress
*As a community member, I want to see how the product is being built and where it is going.*
- **Story 3.1:** View a public roadmap that shows what features are "Launched" and "In Progress."
- **Story 3.2:** See real-time business statistics (Revenue, Users) that signal platform health.
- **Story 3.3:** Submit a feature request directly from the transparency hub.

### Epic 4: Frictionless Handoff (Workspace)
*As an anonymous builder, I want to save my work to a permanent cloud workspace.*
- **Story 4.1:** Click "Save" and see a 1-tap sign-up modal that pre-fills my local draft name.
- **Story 4.2:** Experience a seamless transition where my draft is moved to my cloud account instantly.
- **Story 4.3:** Receive a high-authority "Published" URL that I can share with my audience.
