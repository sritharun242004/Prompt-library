# 03 — Design System
## Clean Creative Specifications · dpecom_platform_03

### 1. The Kit Aesthetic Manifesto
1. **Audience First.** The email capture field is the most important element on the page.
2. **Typography Pairings.** Elegant Serifs (Headings) + Clean Sans (Body) = Trust + Clarity.
3. **Warm Neutrals.** No harsh white. Use `#FAF9F6` (Cream) for backgrounds.
4. **Action Green.** Use `#11AD8D` sparingly but boldly for primary actions.

### 2. Color Palette
- **Primary:** `#11AD8D` (Kit Green) - Conversion CTAs.
- **Secondary:** `#FF6D60` (Coral) - Highlights and Tips.
- **Background:** `#FAF9F6` (Warm Cream) - Main surface.
- **Text:** `#1E293B` (Warm Charcoal) - High readability.

### 3. Typography
- **Headlines:** Playfair Display (or EB Garamond) - `font-serif`, `font-semibold`.
- **Body:** Inter - `font-sans`, `font-normal`.
- **Labels:** `text-xs uppercase tracking-widest font-bold`.

### 4. Component Specs
**The "Kit Card"**
- Centered: `max-w-2xl`
- Padding: `p-12` (Generous)
- Background: `#FFFFFF`
- Radius: `8px`
- Shadow: `shadow-sm`

**The "Tip Button"**
- Shape: `rounded-lg`
- Border: `2px solid #E2E8F0`
- Hover: `border-[#11AD8D] bg-[#F0FAF8]`
- Animation: Heart pulse on click.
