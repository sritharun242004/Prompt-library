STEP 2 — Read 02_Architecture.md
Understand the Next.js 14 stack, the Resend integration for automation, and the Supabase audience-tagging system.

STEP 3 — Read 03_Design.md
Adhere to the "Clean Creative" visual language: serif headings, warm neutrals, and the signature Kit Green (#11AD8D).

STEP 4 — Read 04_Plan.md
Follow the build sequence: Audience Core -> Tip Jar -> Product Logic -> Automated Delivery -> Home Page.

STEP 5 — Read 05_Epics_and_Stories.md
Review the requirements for integrated newsletter opt-ins and frictionless digital sales.

STEP 6 — Read 06_Tasks.md
Execute tasks sequentially. Mark them complete. Wait for review after each task.

STEP 7 — Reference 07_Guide.md
Continuous reference for Resend API patterns, accessibility standards for newsletters, and serif-sans typography pairings.

WORKING RULES:
1. Sequential execution.
2. "Human & Personal" UI is mandatory.
3. Every purchase must trigger an audience-growth event (Resend tag).
4. No aggressive marketing patterns (pop-ups, countdowns).
