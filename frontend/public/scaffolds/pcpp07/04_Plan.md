# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_07

### Phase 1: Canvas & Shell (Week 1)
- Set up Next.js 14 project with the "Atmospheric Minimalism" theme.
- Build the "Minimalist Top-bar" and "Sky-Open" layout container.
- Implement the "AeroHero" slideshow with slow-fade logic.
- **Goal:** A grand, high-impact presence for the landing page.

### Phase 2: Chronology Engine (Week 2)
- Set up Sanity CMS for `Location`, `Story`, and `Media` types.
- Build the dynamic `/stories/[slug]` template with "Time of Day" markers.
- Implement the "Landscape First" grid engine that handles ultra-wide aspect ratios.
- **Goal:** Users can experience a chronological, location-based wedding story.

### Phase 3: The Perspective CMS (Week 3)
- Build the "About" page with aviation-inspired storytelling blocks.
- Implement the technical "Gear & Presets" view using monospace typography.
- Create the "Ambassador" prestige badge component (Fujifilm).
- **Goal:** A high-trust bio section that explains the brand's unique edge.

### Phase 4: International Inquiry Portal (Week 4)
- Build the 1-page "Where are we going?" inquiry form.
- Integrate Supabase for global lead capture and management.
- Implement automated confirmation emails using Resend.
- **Goal:** A high-conversion lead generation portal for global clients.

### Phase 5: Technical Polish & Performance (Week 5)
- Add parallax scaling effects to massive landscape imagery.
- Final SEO and 4G optimization for international destination sites.
- Accessibility audit for high-contrast typography and environmental captions.
- **Goal:** A world-class, production-ready destination wedding platform.
