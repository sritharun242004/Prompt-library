# 04 — Build Plan
## Roadmap & Milestones · lp_platform_01

### Phase 1: Noir Shell & Motion (Week 1)
- Set up Next.js 14 project with the "Tech-Noir" theme and Inter font.
- Build the "Video Hero" engine and the "Sticky Urgency Ticker" marquee.
- Implement the "Global Hub" layout with dark-mode defaults.
- **Goal:** A high-energy, immersive opening presence.

### Phase 2: Speaker Engine & Search (Week 2)
- Set up Sanity.io for `Speaker` and `Track` structured content.
- Build the "Discovery Grid" with Algolia real-time filtering.
- Implement the "Speaker Card" masonry component.
- **Goal:** Users can browse 500+ world-class speakers instantly.

### Phase 3: Multi-Track Schedule Hub (Week 3)
- Build the interactive `ScheduleTimeline` with stage-filtering logic.
- Implement the real-time time-zone formatting and date-tabbing.
- Create the "My Tracks" Zustand store with local persistence.
- **Goal:** A functional, personalized 3-day conference schedule.

### Phase 4: Ticketing Hub & Scarcity (Week 4)
- Build the comparison grid for General, Executive, and Chairperson tiers.
- Integrate Stripe Checkout with support for 2-for-1 discount logic.
- Implement the "Remaining Tickets" real-time scarcity badges.
- **Goal:** High-conversion ticketing funnel is functional.

### Phase 5: Startup Portal & Final Polish (Week 5)
- Build the "Startup Alpha" application landing page and form.
- Final performance audit for 4K video heros and heavy grid LCP.
- Accessibility audit for high-contrast neon colors and marquee speeds.
- **Goal:** A complete, world-class design event digital platform.
