- Set up Next.js 14 project with the "Restrained Baroque" theme.

- Implement the "Vintage Sepia" background theme and serif hierarchy.

- Implement the global store locator and WhatsApp/Email inquiry triggers.
- Create the "Art of Retail" narrative blocks.
- **Goal:** Bridging the digital "gallery" with the physical "temple" of the brand.

- Build the "Heritage" section and the Sabyasachi Art Foundation hub.
