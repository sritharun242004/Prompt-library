# 04 — Build Plan
## Roadmap & Milestones · bw_platform_01

### Phase 1: Luxury Shell & Collection (Week 1)
- Set up Next.js 14 project with the Gold/Montserrat theme.
- Build the "Atelier Nav" (Sticky with centered logo) and full-bleed Hero Engine.
- Implement the "Craft Nav" global strip for the homepage.
- **Goal:** A prestigious homepage presence for current collections.

### Phase 2: Craft Discovery & CMS (Week 2)
- Set up Sanity.io for `Craft` and `Collection` structured content.
- Build the dynamic category routing based on "Artisanal Technique" tags.
- Implement the "Story Block" layout system for narrative sections.
- **Goal:** Users can browse work organized by heritage craft.

### Phase 3: Product Detail & Video Theater (Week 3)
- Build the high-res PDP with "Behind the Craft" video integration.
- Implement the "Detail Zoom" engine for high-density fabric shots.
- Create the "Size-Inclusive" and "Ready-to-Ship" badge logic.
- **Goal:** Deep engagement with garment detail and artisanal process.

### Phase 4: Concierge & Booking Portal (Week 4)
- Build the WhatsApp Inquiry floating component with context-aware triggers.
- Implement the "Book an Appointment" multi-region store locator.
- Build the "Personalized Edit" logic (curated lists based on occasion).
- **Goal:** A high-conversion luxury service layer.

### Phase 5: Sustainability & Final Polish (Week 5)
- Build the dedicated "Grassroot" label hub with community impact data.
- Final SEO and 4G optimization for global-standard performance.
- Final luxury animation polish (springy fades and gold-line expansions).
- **Goal:** A complete, world-class luxury fashion and heritage platform.
