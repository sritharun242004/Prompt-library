# 04 — Build Plan
## Roadmap & Milestones · bw_platform_05

### Phase 1: Nostalgic Shell & Nav (Week 1)
- Set up Next.js 14 project with the Parchment/Green theme and Recoleta font.
- Build the "Sticky Conversion Nav" and the full-bleed Hero Engine.
- Implement the "Homepage Narrative Scroll" layout shell.
- **Goal:** A grand, high-impact presence for the restaurant flagship.

### Phase 2: Dynamic Menu Engine (Week 2)
- Set up Sanity.io for `MenuCategory` and `MenuItem` structured content.
- Build the "Icon-Led Menu" template with sticky category navigation.
- Implement the custom illustrative icon mapping logic.
- **Goal:** Users can browse seasonal dishes with visual charm.

### Phase 3: Reservation Integration (Week 3)
- Build the "Booking Drawer" slide-out using Framer Motion.
- Integrate SevenRooms or OpenTable widget within the drawer container.
- Set up the Zustand state for managing "Active Table" booking sessions.
- **Goal:** A functional, non-disruptive reservation experience.

### Phase 4: Canteen Stories & Shop (Week 4)
- Build the "Canteen Stories" grid and long-form article templates.
- Implement the Merch Shop grid and Gift Card checkout flow.
- Create the "People Behind the Plate" team section.
- **Goal:** Integrated storytelling and secondary revenue streams.

### Phase 5: Glocal Polish & Performance (Week 5)
- Optimize high-vibrancy food imagery for mobile LCP.
- Final luxury animation polish (springy drawer slides and staggered reveals).
- Final SEO and Page Speed audit for global culinary travelers.
- **Goal:** A complete, world-class restaurant digital platform.
