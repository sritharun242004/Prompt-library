# 04 — Build Plan
## Roadmap & Milestones · lp_platform_03

### Phase 1: Invitation Shell & Layout (Week 1)
- Set up Next.js 14 project with the "Minimalist Elegance" theme and EB Garamond font.
- Build the "Centered Landing" shell and the responsive "Cover Image" component.
- Implement the "Invitation Typography" hierarchy across headers and body.
- **Goal:** A beautiful, breathable landing page structure.

### Phase 2: Avatar Stack & Social Proof (Week 2)
- Set up Sanity.io for `Event` and `Attendee` structured content.
- Build the "Overlapping Avatar Stack" component with dynamic Z-indexing.
- Implement the "Social Validation" strip ("Who's Going") below the title.
- **Goal:** Users experience immediate community trust on entry.

### Phase 3: Event Detail Template (Week 3)
- Build the "Logistics Block" with integrated map previews and calendar triggers.
- Create the "Host Profile" component with "Follow" button logic.
- Implement the "Event Card" grid for community discovery pages.
- **Goal:** A functional, information-rich event browsing experience.

### Phase 4: Registration & Auth (Week 4)
- Integrate Clerk for social-first, frictionless registration.
- Build the "One-Tap Sign-up" modal with minimal input fields.
- Implement the server-side registration confirmation and email triggers.
- **Goal:** High-conversion registration flow is functional.

### Phase 5: Community Hub & Polish (Week 5)
- Build the "Community Profile" hub with recurring event calendars.
- Final luxury animation polish (soft fades and centered entries).
- Accessibility audit for avatar alt-text and high-contrast typography.
- **Goal:** A complete, world-class social event platform.
