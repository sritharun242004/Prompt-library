# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_13

### Phase 1: Typography & Reading Shell (Week 1)
- Set up Next.js 14 project with the "Analytic Austerity" theme.
- Build the "Typography Engine" (Global serif variables and 680px resets).
- Implement the "Minimal Nav" and "Unified Feed Shell."
- **Goal:** A perfect, distraction-free environment for strategic analysis.

### Phase 2: Analysis Feed & Diagram SVG (Week 2)
- Set up the CMS logic for long-form essays and "Daily Update" excerpts.
- Build the "Thompson Diagram" SVG component system.
- Implement the chronological homepage feed with "Analytic Hierarchy."
- **Goal:** A working news-stream with signature visual signatures.

### Phase 3: Passport Delivery Core (Week 3)
- Build the Edge Function for dynamic, secure RSS/Podcast feed generation.
- Implement the "Member Dashboard" for managing personalized feed tokens.
- Set up the security logic to track and revoke suspicious feed usage.
- **Goal:** Secure, one-to-one content delivery across all channels.

### Phase 4: Concept Index & Strategic Search (Week 4)
- Build the "Concept Index" grid and framework detail pages.
- Integrate Algolia for hierarchical search (Frameworks > Companies > Articles).
- Create the "Foundational Threads" (linking concepts to years of work).
- **Goal:** Ease of discovery for a decade of strategic knowledge.

### Phase 5: Plus Bundling & Final Polish (Week 5)
- Build the "Stratechery Plus" multi-stream subscription logic.
- Integrate Stripe for complex bundling and member portal access.
- Final SEO and Performance audit (Focus on TBT - Total Blocking Time).
- **Goal:** A complete, world-class strategic analysis and subscription business.
