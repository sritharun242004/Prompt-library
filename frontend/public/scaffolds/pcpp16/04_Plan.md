# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_16

### Phase 1: Garden Infrastructure (Week 1)
- Set up Next.js 14 project with the "Minimalist Builder" theme.
- Build the "Typography Engine" (Global Tiempos/Inter pairings).
- Implement the "Searchable Index" shell and the Algolia search bar.
- **Goal:** A fast-loading, scannable content hub.

### Phase 2: Note & Article Engine (Week 2)
- Set up Sanity.io for `Note` and `Article` content types.
- Build the "Nat's Notes" template with the summary box and rating UI.
- Implement the chronological feed with shared category filtering.
- **Goal:** Users can browse and read through the 500+ entry archive.

### Phase 3: Course Delivery Shell (Week 3)
- Build the `CoursePlayer` layout (Sidebar curriculum + Video Main).
- Integrate Mux Player for secure, high-bitrate video streaming.
- Implement the MDX lesson renderer with support for code and resources.
- **Goal:** A professional-grade learning environment.

### Phase 4: Membership & Pro Wall (Week 4)
- Integrate Stripe for Course and "Builder Pro" subscription sales.
- Build the "Pro Dashboard" with integrated workshop calendars and links.
- Implement the `AccessGuard` server components for protected content.
- **Goal:** Functional recurring revenue and premium member access.

### Phase 5: Discovery & Final Speed (Week 5)
- Build the "Monday Medley" newsletter landing page and lead magnets.
- Implement the "Related Notes" and "Course Upsell" recommendation logic.
- Final performance and accessibility audit (Targeting sub-1s LCP).
- **Goal:** A complete, world-class personal brand and educational business.
