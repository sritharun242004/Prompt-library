# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_18

### Phase 1: Marketplace & Grid (Week 1)
- Set up Next.js 14 project with the "Professional Mastery" theme and Inter font.
- Build the "Atelier Nav" (Sticky with domain categories) and the Marketplace Grid engine.
- Implement the "Instructor Card" with pedigree-first visual hierarchy.
- **Goal:** A scannable, high-authority marketplace shell.

### Phase 2: Curriculum Engine & CMS (Week 2)
- Set up Sanity.io for `Instructor`, `Course`, and `Cohort` structured content.
- Build the dynamic `/courses/[slug]` template with weekly accordions.
- Implement the "Urgency Hook" for real-time start-date calculations.
- **Goal:** Users can browse detailed syllabi with live enrollment cues.

### Phase 3: Enrollment & Expense Suite (Week 3)
- Build the multi-step enrollment form and the Stripe-integrated payment modal.
- Implement the "Expense This Course" utility for professional reimbursement.
- Create the "Transformations" outcomes list for course heros.
- **Goal:** Functional high-AOV conversion and corporate purchase support.

### Phase 4: Student Dashboard & Sync (Week 4)
- Build the `StudentPortal` for cohort-specific material and peer networking.
- Set up the Supabase logic for `enrollment` verification and `progress` tracking.
- Implement the "Calendar Sync" feature for live cohort sessions.
- **Goal:** End-to-end student experience post-purchase.

### Phase 5: Search & Final Optimization (Week 5)
- Integrate Algolia for high-speed marketplace search and domain filtering.
- Final performance audit for large-scale grid LCP optimization.
- Accessibility audit for high-contrast typography and portrait labels.
- **Goal:** A complete, world-class cohort-based learning platform.
