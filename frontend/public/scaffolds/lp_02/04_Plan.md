# 04 — Build Plan
## Roadmap & Milestones · lp_platform_02

### Phase 1: Enterprise Shell & Nav (Week 1)
- Set up Next.js 14 project with the "Enterprise Modern" theme and Inter font.
- Build the "Glass Nav" (Sticky with Orange CTA) and the Thematic Hub shell.
- Implement the "Innovation Visual" hero background.
- **Goal:** A prestigious, high-authority opening presence.

### Phase 2: Thematic Pillar Engine (Week 2)
- Set up Sanity.io for `Theme` and `Pillar` structured content.
- Build the icon-driven pillar grid with high-contrast text.
- Implement the "Tech Café" horizontal carousel for startup showcases.
- **Goal:** Users can understand the forum's strategic focus areas.

### Phase 3: Multi-View Agenda Hub (Week 3)
- Build the complex `AgendaHub` timeline with `LIST` vs `VENUE` view logic.
- Implement the Zustand state for "My Schedule" and session filtering.
- Create the "Venue Grid" CSS layout for physical event planning.
- **Goal:** A functional, high-density 3-day conference schedule.

### Phase 4: Speaker Directory & Discovery (Week 4)
- Build the authoritative speaker grid with Algolia real-time filtering.
- Implement the "Speaker Profile" modal with linked session itineraries.
- Create the "Success Metrics" counting animations (e.g., "100+ Speakers").
- **Goal:** Established star-power and industry authority.

### Phase 5: App Sync & Final Polish (Week 5)
- Build the "Konnect Sync" persistent strip with QR code logic.
- Final performance audit for large-scale grid and glassmorphism LCP.
- Accessibility audit for high-contrast colors and interactive timeline states.
- **Goal:** A complete, world-class enterprise event digital platform.
