# 04 — Build Plan
## Roadmap & Milestones · lp_platform_11

### Phase 1: Neutral Shell & Foundation (Week 1)
- Set up Next.js 14 project with the "Designer-Grade" theme and Inter font.
- Build the "Atelier Nav" (Sticky with minimal links) and the Gallery Shell.
- Implement the "White Paper" background theme and 1px border tokens.
- **Goal:** A prestigious, high-fidelity gallery container.

### Phase 2: Masonry Grid & CMS (Week 2)
- Set up Sanity.io for `Template`, `Creator`, and `Category` structured content.
- Build the dynamic masonry grid component using CSS Grid or a library.
- Implement the "Template Card" with price-badge and creator metadata.
- **Goal:** Users can browse the visual catalog with high-fidelity pop.

### Phase 3: Discovery Engine & Filters (Week 3)
- Integrate Algolia for near-instant template discovery and filtering.
- Build the "Filter Bar" with category-pill logic and search inputs.
- Create the "Zustand" store for managing active discovery states.
- **Goal:** A functional, high-speed discovery interface.

### Phase 4: Detail Hub & Live Preview (Week 4)
- Build the `TemplateDetail` modal with page-by-page scrolling snapshots.
- Implement the "Live Preview" portal (external link or iframe).
- Create the "Remix Trigger" logic for 1-click cloning.
- **Goal:** High-conversion detail discovery is live.

### Phase 5: Creator Integration & Polish (Week 5)
- Build the creator profile pages and "Other works" carousels.
- Final performance audit for heavy image-grid LCP (Target <1s).
- Accessibility audit for high-contrast colors and grid navigation.
- **Goal:** A complete, world-class template marketplace flagship.
