# 04 — Build Plan
## Career Acceleration & Tech Upskilling Platform · business_platform_04

---

### Overview

| Phase | Name | Duration | Goal |
|-------|------|----------|------|
| 0 | Foundation | 1 week | IA, schema, CSS token setup, navigation shell |
| 1 | Placement Proof | 1 week | proof band, alumni story cards, company logo grid |
| 2 | Program Discovery | 1 week | program track pages, curriculum breakdowns, outcome data |
| 3 | Mentor Credibility | 3 days | mentor grid, profile cards, 1:1 model explanation |
| 4 | Scholarship Flow | 1 week | assessment steps, tier reveal, enrollment prompt |
| 5 | Enrollment Funnel | 2 weeks | demo booking form, inquiry form, fee/EMI section, CRM |
| 6 | QA and Launch | 1 week | analytics, a11y, mobile performance, launch checklist |

---

### Phase Gates

- [ ] proof-context gate: every salary and placement metric has cohort year and program label
- [ ] mentor data gate: all mentor profiles show current company, role, and experience
- [ ] form reliability gate: demo booking and inquiry forms validate and submit without error
- [ ] scholarship gate: assessment scores server-validated; tier reveal triggers correctly
- [ ] analytics gate: all funnel events fire with program and background context payload
- [ ] accessibility gate: WCAG AA pass on all interactive elements
- [ ] performance gate: Lighthouse 95+ on mobile for homepage and demo route

---

### Cut Order

Cut if time is short (in order):
1. secondary alumni story variants and video testimonials
2. ISA (income share) option page (defer to post-launch)
3. full mentor directory page (keep featured mentor section on homepage)

Never cut:
- placement proof band with company logos
- program track pages with curriculum and outcome data
- demo booking form and inquiry form
- fee/EMI transparency section
- scholarship assessment and tier reveal flow
