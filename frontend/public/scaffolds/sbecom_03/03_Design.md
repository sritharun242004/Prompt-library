# 03 — Design System
## Playful & Premium Specifications · sbecom_platform_03

### 1. The Glow Manifesto
1. **Visual over Verbal.** If an option can be represented by a beautiful icon or photo (e.g., hair type), do it.
2. **Softness is Luxury.** Avoid sharp 90-degree corners. Use 16px-24px radii for a "pillowy" premium feel.
3. **Pastel Power.** Use Soft Peony (#F9A8D4) for main CTAs. It should feel inviting and high-end.
4. **Anticipatory Motion.** Use "Staggered reveals" for product samples to mimic the unboxing experience.

### 2. Color Palette
- **Primary:** `#F9A8D4` (Soft Peony) - Call to Action.
- **Secondary:** `#FB923C` (Warm Coral) - Special offers & Best Value.
- **Background:** `#FFFFFF` (White) / `#FFF7ED` (Warm Shell for sections).
- **Surface:** `#FDF2F8` (Softest Pink for cards).
- **Text:** `#374151` (Deep Slate - softer than black).

### 3. Typography
- **Headlines:** Montserrat (Weight 600) - `font-semibold`, `tracking-tight`.
- **Body:** Inter (Weight 400) - `font-normal`, `leading-relaxed`.
- **Accents:** Italic Serif (e.g., Playfair) used sparingly for "Editor's Notes."

### 4. Component Specs
**The "Visual Quiz Input"**
- Outline: `2px solid transparent`.
- Selected State: `border-[#F9A8D4] bg-[#FDF2F8]`.
- Content: Centered Image -> Title text below.

**The "Plan Selection Card"**
- Elevation: `shadow-md` on the recommended plan.
- Badge: "Best Value" ribbon in Warm Coral at top-right.
- Layout: Duration -> Savings -> Price per month -> "Select" (Pink).
