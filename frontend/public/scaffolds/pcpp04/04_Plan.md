# 04 — Build Plan
## Roadmap & Milestones · pcpp_platform_04

### Phase 1: Universe Core & Multi-tenancy (Week 1)
- Set up Next.js 14 project with Middleware for domain-based routing.
- Build the "Central Hub" with large-scale project portal cards.
- Implement the "Universe Switcher" global component.
- **Goal:** A working multi-project navigation core.

### Phase 2: Narrative Sequence Engine (Week 2)
- Build the `StoryBlock` system (Single, Pair, Triptych, Text).
- Implement the long-scroll essay template with `IntersectionObserver` for sequences.
- Set up the data model for manual image sequencing.
- **Goal:** Users can experience a high-end photo essay.

### Phase 3: Project Planets & Navigation (Week 3)
- Build the project-specific landing pages and "Journey" sidebars.
- Implement the "The Essay," "The Series," and "The Press" sub-routing.
- Create the "Back to Top" and "Story Progress" indicators.
- **Goal:** Each project planet is fully navigable and immersive.

### Phase 4: Journal & Diary CMS (Week 4)
- Set up the MDX or Sanity CMS for long-form diaries and interviews.
- Build the "Journal" archive and single-post templates.
- Implement "Linked Sequences" (Blog post -> Related project sequence).
- **Goal:** Deep integration of text and imagery.

### Phase 5: Polish & Sequence Transitions (Week 5)
- Build the portal-style transitions between universes.
- Add staggered scroll-triggered reveals for sequence pairings.
- Final SEO, performance, and multi-domain configuration audit.
- **Goal:** A complete, world-class photography and sociology universe.
