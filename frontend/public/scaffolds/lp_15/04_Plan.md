# 04 — Build Plan
## Roadmap & Milestones · lp_platform_15

### Phase 1: Electric Shell & Theme (Week 1)
- Set up Next.js 14 project with the "Electric Youth" theme and Montserrat/Inter fonts.
- Build the "Atelier Nav" (Visual categories + Red CTA) and the Pitch Black shell.
- Implement the "High-Saturation" color tokens and CSS resets.
- **Goal:** A bold, high-energy entry portal container.

### Phase 2: Launchpad Slider & CMS (Week 2)
- Set up Sanity.io for `Launch`, `Personality`, and `Product` schemas.
- Build the full-bleed `HeroSlider` with celebrity-promotional story support.
- Implement the "New Launch" badge logic and entry animations.
- **Goal:** Users experience a world-class product drop on entry.

### Phase 3: FOMO Engine & Daily Deals (Week 3)
- Build the `CountdownTimer` and `FlashSaleTicker` components.
- Implement the high-density "Daily Deals" grid with massive percentage labels.
- Set up real-time stock-count syncing for "Selling Fast" notifications.
- **Goal:** Established high-velocity urgency across the hub.

### Phase 4: Celebrity Hub & Tribe (Week 4)
- Build the "Celebrity Grid" component linking ambassadors to collections.
- Implement the "Shop by Vibe" lifestyle discovery blocks.
- Create the "boAt Rewards" gamified sign-up widget.
- **Goal:** Immersive community discovery and loyalty acquisition is functional.

### Phase 5: Mobile Density & Polish (Week 5)
- Optimize all product grids for horizontal swiping (Swiper.js integration).
- Final performance audit for heavy celebrity imagery LCP (Target <1.5s).
- Accessibility audit for high-contrast colors and screen-reader timers.
- **Goal:** A complete, world-class high-energy lifestyle flagship.
