# 03 — Design System
## Modern SaaS Specifications · dpecom_platform_02

### 1. The SaaS Polish Manifesto
1. **Whitespace is a feature.** Avoid clutter; give data room to breathe.
2. **Shadows are soft.** Use multi-layered, low-opacity shadows for depth.
3. **Corners are friendly.** 12px-16px radii on all cards and buttons.
4. **Trust via Clarity.** Clear labels, helpful tooltips, and high-contrast typography.

### 2. Color Palette
- **Primary Accent:** `#FFEF5E` (Lemon Yellow) - Action items.
- **Secondary Accent:** `#7047EB` (Vivid Purple) - Structure/Links.
- **Success:** `#10B981` (Emerald) - Revenue growth.
- **Background:** `#F9FAFB` (Soft Grey) - Dashboard background.
- **Surface:** `#FFFFFF` (White) - Cards and Modals.

### 3. Typography
- **Headlines:** Plus Jakarta Sans - `font-bold`, `tracking-tight`.
- **Body:** Inter - `font-normal`, `leading-normal`.
- **Financials:** JetBrains Mono - For MRR values and license keys.

### 4. Component Specs
**The "SaaS Card"**
- Background: `#FFFFFF`
- Border: `1px solid #E5E7EB`
- Radius: `16px`
- Shadow: `shadow-sm` (subtle depth)

**Buttons**
- Primary: `bg-[#FFEF5E] text-slate-900 rounded-xl px-5 h-11 font-semibold hover:bg-yellow-300`
- Secondary: `bg-slate-900 text-white rounded-xl px-5 h-11 font-semibold`
