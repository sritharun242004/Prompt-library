Changelog

Updates, fixes, and new features.

May 13, 2024v2.4.0Feature

Command palette and keyboard shortcuts

Press Cmd+K to open the command palette. Search issues, navigate to any page, or run actions — all without touching the mouse. Full keyboard navigation with arrow keys and Enter.

May 6, 2024v2.3.0Feature

Real-time issue updates

Issue changes from teammates now appear instantly without a page refresh. Powered by Supabase Realtime. Works across tabs.

April 29, 2024v2.2.1Fix

Issue filter performance

Fixed a performance regression in the issue list when applying multiple filters simultaneously. List now responds in under 50ms.

April 22, 2024v2.2.0Feature

Cycle progress tracking

Cycles now show a real-time progress bar and velocity metric. See how many issues your team is completing per day.
