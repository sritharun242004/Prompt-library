About

Sanya Mehta · Mumbai, India

I'm a self-taught developer who fell in love with the web. I started with HTML and CSS in my college hostel room, moved on to JavaScript, and never looked back. These days I work mainly with React, Node.js, and a healthy obsession with CSS. I believe the best way to understand something deeply is to write about it — which is why I've been writing technical articles since 2019.

I'm particularly interested in web fundamentals, performance, and making JavaScript concepts approachable for beginners. I write without jargon where possible, because clarity is a form of respect.
