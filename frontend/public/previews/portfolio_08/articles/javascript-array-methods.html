JavaScript Array Methods

September 14, 20222 min read
  • javascript
  • fundamentals

JavaScript arrays come loaded with powerful methods. Mastering them makes your code cleaner, more declarative, and easier to reason about.

The Big Three: map, filter, reduce

These three handle the vast majority of data transformation tasks.

map — Transform each element

const prices = [10, 25, 50]
const withTax = prices.map(p => p * 1.18)
// [11.8, 29.5, 59]

const users = [{ name: 'Alice' }, { name: 'Bob' }]
const names = users.map(u => u.name)
// ['Alice', 'Bob']

filter — Keep matching elements

const numbers = [1, 2, 3, 4, 5, 6]
const evens = numbers.filter(n => n % 2 === 0)
// [2, 4, 6]

const activeUsers = users.filter(u => u.active)

reduce — Accumulate to a single value

const total = [10, 25, 50].reduce((sum, n) => sum + n, 0)
// 85

// Group by category
const grouped = products.reduce((acc, product) => {
  const key = product.category
  if (!acc[key]) acc[key] = []
  acc[key].push(product)
  return acc
}, {})

Finding Elements

const arr = [3, 7, 12, 5]

arr.find(n => n > 10)     // 12 — first match
arr.findIndex(n => n > 10) // 2 — index of first match
arr.includes(7)            // true
arr.indexOf(7)             // 1
arr.some(n => n > 10)     // true — at least one match
arr.every(n => n > 0)     // true — all match

Sorting

const nums = [10, 1, 5, 100]
nums.sort((a, b) => a - b)  // ascending: [1, 5, 10, 100]
nums.sort((a, b) => b - a)  // descending: [100, 10, 5, 1]

const people = [{ name: 'Zara' }, { name: 'Alice' }]
people.sort((a, b) => a.name.localeCompare(b.name))

Note: .sort() mutates the original array. Use [...arr].sort() to sort a copy.

Flattening

const nested = [[1, 2], [3, 4], [5]]
nested.flat()       // [1, 2, 3, 4, 5]
nested.flat(2)      // deeply nested — flat up to depth 2

// flatMap — map then flatten one level
const sentences = ['hello world', 'foo bar']
sentences.flatMap(s => s.split(' '))
// ['hello', 'world', 'foo', 'bar']

Slicing and Splicing

const arr = [1, 2, 3, 4, 5]

arr.slice(1, 3)         // [2, 3] — non-mutating
arr.splice(1, 2)        // removes 2 elements from index 1 — mutating!

// Immutable "remove item at index"
const removed = [...arr.slice(0, 2), ...arr.slice(3)]

Chaining Methods

const result = orders
  .filter(o => o.status === 'completed')
  .map(o => o.total)
  .reduce((sum, total) => sum + total, 0)

Summary

  • map transforms, filter selects, reduce accumulates
  • find / findIndex / some / every for searching
  • Always sort copies ([...arr].sort()) unless mutation is intentional
  • flat and flatMap handle nested arrays
  • Chain methods for readable data pipelines
