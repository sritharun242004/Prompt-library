CSS Variables Deep Dive

November 5, 20231 min read
  • css
  • design-systems
  • frontend

CSS custom properties (often called CSS variables) have been available in all major browsers since 2017, yet many developers still underuse them. They're far more powerful than preprocessor variables.

Basic Syntax

:root {
  --color-brand: #ec4899;
  --spacing-md: 1rem;
  --font-heading: 'Outfit', sans-serif;
}

.button {
  background: var(--color-brand);
  padding: var(--spacing-md);
  font-family: var(--font-heading);
}

Fallback Values

var() accepts a fallback as the second argument:

.card {
  color: var(--color-text, #333);
  background: var(--card-bg, var(--color-bg, white));
}

Theming with Data Attributes

:root {
  --bg: hsl(40deg 33% 97%);
  --text: hsl(210deg 12% 16%);
}

[data-theme="dark"] {
  --bg: hsl(210deg 12% 13%);
  --text: hsl(40deg 20% 90%);
}

Toggle the theme in JavaScript:

document.documentElement.setAttribute('data-theme', 'dark')

Computed and Inherited Values

Unlike Sass variables, CSS custom properties are part of the cascade. They inherit through the DOM and can be scoped to any element:

.card {
  --card-radius: 0.5rem;
  border-radius: var(--card-radius);
}

.card--large {
  --card-radius: 1rem; /* overrides locally */
}

JavaScript Integration

// Read a variable
const style = getComputedStyle(document.documentElement)
const brand = style.getPropertyValue('--color-brand').trim()

// Set a variable
document.documentElement.style.setProperty('--color-brand', '#06b6d4')

This opens up dynamic theming, live colour pickers, and animation at the CSS layer.

Animations with @property

Register a custom property to animate it:

@property --gradient-stop {
  syntax: '<percentage>';
  inherits: false;
  initial-value: 0%;
}

.button {
  background: linear-gradient(to right, #ec4899 var(--gradient-stop), #8b5cf6 100%);
  transition: --gradient-stop 0.3s;
}

.button:hover {
  --gradient-stop: 50%;
}

Summary

  • CSS variables live in the cascade — they inherit and can be scoped
  • Use :root for global tokens, element selectors for local overrides
  • [data-theme] is the cleanest theming approach
  • JavaScript can read and write custom properties at runtime
  • @property enables animating custom properties
