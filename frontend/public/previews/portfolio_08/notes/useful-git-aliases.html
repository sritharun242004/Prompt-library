Useful Git aliases I use daily

March 1, 2024
  • git
  • tools
  • workflow

After years of using Git, these aliases have become muscle memory. Add them to your ~/.gitconfig:

[alias]
  st = status -sb
  lg = log --oneline --graph --decorate --all
  undo = reset HEAD~1 --mixed
  unstage = restore --staged
  aliases = config --get-regexp ^alias\\.
  pushup = push -u origin HEAD
  recent = for-each-ref --sort=-committerdate refs/heads/ --format='%(refname:short)'

The ones I use most:

  • git st — short status is almost always enough
  • git lg — the graph view makes branch history readable
  • git undo — undo last commit but keep changes staged (safer than reset --hard)
  • git pushup — push new branch and set upstream in one command

git recent shows your branches sorted by most recently used — great when you work across many features.
