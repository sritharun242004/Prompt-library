Code and Media Are Permissionless Leverage

2024-09-227 tweets

  1. There are three broad classes of leverage: capital, labor, and products with no marginal cost of replication.

  2. Capital and labor are permissioned leverage. Everyone is chasing them.

  3. Code and media are permissionless leverage. They are the leverage of the new rich.

  4. You can write a book, publish code, record a video—once—and have it work for you forever. No boss required.

  5. An army of robots is freely available to anyone who can write code. This is a massive opportunity for wealth creation.

  6. If you are not building or buying equity in a business, you are selling your time. You are capped at how many hours you can sell.

  7. Combine specific knowledge + accountability + leverage and you will get outsized returns in the modern economy.
