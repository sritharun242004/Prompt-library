Spaces — one browser, every version of you.

Stop switching browser profiles. Spaces give each side of your life — work, personal, research, side projects — its own isolated environment. Separate tabs, separate history, separate identity. All in one window.

  • Unlimited Spaces on all plans
  • Per-Space tab isolation and history
  • Custom theme per Space
  • Keyboard shortcut to switch instantly
  • Pinned sites per Space
  • Shared bookmarks across Spaces (optional)
Spaces — organized life

Boosts — customize the internet.

Every website you visit can be remade exactly how you want it. Hide distracting elements, inject custom CSS, add JavaScript shortcuts. Boosts are shareable — find thousands of community Boosts in the gallery.

  • Visual point-and-click editor
  • CSS injection with live preview
  • JavaScript shortcut execution
  • Public Boosts gallery
  • Import/export Boosts
  • Per-domain or global Boosts
Boosts — website customization

Iris Intelligence — AI that sees your screen.

No more copy-paste into ChatGPT. Iris reads the active page and answers questions in context. Summarize a long article, translate a foreign-language site, explain jargon, or just ask anything about what you're reading.

  • Reads any visible page content
  • In-browser chat panel (no new tab)
  • Smart tab summaries on hover
  • Inline explanations and translations
  • Works with all websites
  • Powered by latest frontier models
Iris AI — in-browser intelligence

Split View — two pages, one flow.

Research + writing. Dashboard + docs. Two sources side-by-side, no extension required. Supports any URL, any web app. 3-way split on wide screens. Drag the divider to resize.

  • Any URL can be split
  • Horizontal and vertical layouts
  • Drag-to-resize divider
  • 3-way layout on wide screens
  • Independent scroll and interaction
  • Persistent across sessions
Split View — parallel browsing

Command Bar

Hit ⌘T to search, navigate, switch Spaces, or run Boosts — all from the keyboard.

Open sites, search history, switch Spaces, trigger Boosts, manage tabs — one shortcut, infinite power.

Ad Blocking

Built-in. No setup. No extension. Ads and trackers blocked natively.

Network-level blocking means faster page loads, less CPU usage, and more privacy without any configuration.

Picture-in-Picture

Pop out any video. Keep watching while you work.

Supports YouTube, Twitch, Loom, Zoom, Meet — any video element. Floats above all windows, remembers position.

Unlimited Themes

Every Space has its own look. Community gallery with hundreds of themes.
